			    	                 APPRENTICE_ARM_PROJECT


	Apprentice Arm is a real-time, hand-gesture-controlled 6-Degree-of-Freedom (6-DOF) robotic arm system designed to enable intuitive and contactless human-robot interaction. The system leverages computer vision and machine learning techniques to detect and interpret natural hand gestures captured through a standard webcam. The core of the proposed system is based on MediaPipe Hands, which extracts 21 three-dimensional hand landmarks and translates them into precise servo motor movements, including base rotation, shoulder lift, elbow bend, wrist pitch, wrist roll, and gripper control. To improve stability and responsiveness, the system incorporates signal conditioning techniques such as Exponential Moving Average (EMA) smoothing and deadband filtering, ensuring smooth and jitter-free motion. The architecture integrates a Python Flask backend for real-time processing, serial communication with an Arduino UNO, and a PCA9685 PWM servo driver for accurate motor control. Additionally, a modern web-based dashboard is developed using HTML, CSS, JavaScript, and Three.js to provide live video streaming, a 3D digital twin of the robotic arm, and manual control interfaces. To enhance automation and intelligence, the system introduces a machine learning module based on K-Means clustering that analyzes recorded task sequences and generates smart presets representing frequently used arm positions. Furthermore, a task recording and playback mechanism is implemented to allow users to store and reproduce complex motion sequences, reducing manual effort and improving operational efficiency. Experimental evaluation demonstrates that the system achieves low latency (~60 ms), real-time performance (~30 FPS), and high usability, making it suitable for educational, research, and industrial applications. The proposed system offers a low-cost, scalable, and user-friendly solution for gesture-based robotic control, with potential extensions toward advanced automation and human-robot collaboration
