# APPRENTICE ARM PROJECT

## Overview
Apprentice Arm is a real-time hand gesture controlled 6-DOF robotic arm system designed for intuitive human-robot interaction.

The system uses:
- MediaPipe Hands
- Python Flask Backend
- Arduino UNO
- PCA9685 Servo Driver
- OpenCV
- Machine Learning

## Features
- Real-time gesture recognition
- 6-DOF robotic arm movement
- Web dashboard control
- Task recording and playback
- Smart preset generation

## Technologies Used
### Software
- Python
- Flask
- OpenCV
- MediaPipe
- HTML
- CSS
- JavaScript
- Three.js

### Hardware
- Arduino UNO
- PCA9685 PWM Driver
- MG996R Servos
- SG90 Servos
- Webcam
